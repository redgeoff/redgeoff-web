# redgeoff-web

## Run locally

- `git clone git@github.com:redgeoff/redgeoff-web`
- `cd redgeoff-web`
- `git submodule update --init --recursive`
- `brew install hugo`
- `hugo server`

## Create new post

e.g. `hugo new posts/everyone-can-make-software.md`
